Mower
=====

Dependencies
------------
* JDK 1.6
* Maven 3
* SLF4J + LogBack
* TestNG
* FestAsserts

Compile & Run
-------------

    git clone https://github.com/mdulac/mower.git
    cd mower
    mvn clean package
    java -jar target/mowitnow-0.0.1-SNAPSHOT.one-jar.jar src/main/resources/mower.txt
